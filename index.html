How to Use

  • Click a mob to see which biomes it spawns in (biomes will glow)
  • Click a biome to see which mobs spawn there (background changes, mobs glow)
  • Hover over any tile to see detailed stats
  • Use filters to narrow down mobs by behavior, spawn type, health, or damage
  • Toggle dimensions to show/hide Overworld, Nether, or End biomes
  • Click outside any tile to reset the view

Custom Visualization Category: COMBINATORIAL

This visualization is classified as COMBINATORIAL because it combines multiple existing visualization techniques and interaction paradigms into a unique, integrated exploration interface:

Combined Techniques:

  1. Grid/Tile Layout: A standard categorical visualization technique for displaying discrete entities in a spatial arrangement
  2. Network/Relational Visualization: Implicitly shows connections between mobs and biomes through the fauna-geography relationships (FaunaGeography.csv)
  3. Coordinated Multiple Views: Two linked sections where interactions in one affect the display in the other
  4. Multi-dimensional Filtering: Standard filtering technique allowing users to explore data by multiple attributes simultaneously
  5. Semantic Focus+Context: Dimming and highlighting creates focus while maintaining context of all entities
  6. Environmental Theming: Using background texture transformation to provide contextual immersion

What Makes It Custom:

The innovation lies in the bidirectional relationship exploration combined with environmental immersion. The system allows two distinct exploration modes:

  1. Mob-centric exploration: "Where does this creature live?" - Select a mob to see its habitat distribution
  2. Biome-centric exploration: "What lives here?" - Select a biome to see its fauna, with the mob section transforming to match the biome's environment

The background transformation feature is particularly novel. When a biome is selected, the mob section's background changes to that biome's texture, creating an immersive "You Are Here!" experience. This goes beyond typical linked visualizations by adding an environmental storytelling layer.

Additionally, the multi-attribute encoding (color for behavior, borders for dimension, glow for relationships) creates a dense information display suitable for a museum exhibit where visitors can explore at their own pace.
